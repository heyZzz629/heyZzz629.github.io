## 🎯 游戏目标
扮演一名信息学竞赛选手（OIer），在**3年高中生活**中：
- 提升算法能力，争取在 NOIP 拿到省一/国集资格
- 平衡文化课、心态、体力、人际关系
- 解锁成就，书写你的 OI 故事

---

## 📊 状态面板说明

| 属性 | 含义 | 注意事项 |
|------|------|----------|
| 💻 算法能力 | 你的 OI 水平，决定比赛成绩 | >85 可冲省选，>95 有望国集 |
| 📚 文化课 | 学业成绩，避免班主任谈话 | <40 会触发"班主任谈话"事件 |
| 🧠 心态 | 心理状态，影响发挥稳定性 | <20 会触发"RE/MLE 崩溃"事件 |
| 💪 体力 | 身体状态，归零会强制住院 | 每回合自动 -2，注意休息 |
| 👥 人际关系 | 与同学/学长的关系 | 影响随机事件和求助成功率 |
| 💰 零花钱 | 购买书籍、饮料等道具 | 可通过兼职赚取 |

## 🎮 如何操作

### 基础操作
1. **选择行动**：点击下方 4 个行动按钮之一
2. **查看日志**：在"游戏日志"标签查看剧情发展
3. **切换标签**：点击顶部标签切换【日志/事件/日记/统计】

### 快捷键 ⚡
```
数字键 1-4 → 快速选择对应行动
```

### 行动类型示例
```
🔹 刷模板题 → 稳定提升算法，消耗少量体力
🔹 挑战难题 → 高风险高回报，可能 AC 也可能 WA
🔹 参加模拟赛 → 检验实力，获得声望
🔹 上文化课 → 提升文化课，避免班主任谈话
🔹 休息放松 → 恢复体力和心态
🔹 帮助同学 → 提升人际，巩固知识
🔹 购买道具 → 使用金钱获得增益
🔹 通宵学习 → 极限提升算法，但极度消耗体力
```

## ⚙️ 游戏机制

### 时间系统
```
第 X 年 → 第 Y 月 → 第 Z 周 → 白天/夜晚
```
- 每选择一次行动 = 推进半回合
- 白天→夜晚→白天 = 推进一周
- 12 个月 = 推进一年，共 3 年游戏时间

### 随机事件 🎲
游戏会随机触发 OI 特色事件：
```
🐛 Bug 之夜：深夜调试，可能找到 Bug 也可能心态崩
👨‍🏫 班主任突袭：文化课太低会被谈话
🏆 AC 时刻：首次 AC 难题，获得成就感
🍜 食堂惊喜：吃到好吃的，恢复状态
⚡ 停电了：代码没保存，损失进度
```

### 成就系统 🏆
```
✅ 初次 AC：第一次成功 AC
👑 机房之王：模拟赛获得第一名
💻 算法高手：算法能力达到 80
🌟 传奇 OIer：算法能力达到 100
⬡ 六边形战士：全属性 80+
```

## 💡 新手攻略

### 🌱 前期（第 1 年）
```
1. 优先刷「模板题」稳定提升算法
2. 偶尔上文化课，保持学术 >40
3. 多「帮助同学」积累人际
4. 体力<30 时及时「休息」
```

### 🚀 中期（第 2 年）
```
1. 尝试「挑战难题」快速提升
2. 参加「模拟赛」检验实力
3. 购买「算法书」突破瓶颈
4. 注意心态管理，避免崩溃
```

### 🏁 后期（第 3 年）
```
1. 算法>85 冲刺省一
2. 平衡各项属性争取完美结局
3. 解锁剩余成就
```

---

## ⚠️ 注意事项

```
❗ 体力 = 0 → 强制住院一回合，无法行动
❗ 文化课 < 40 → 班主任谈话，心态 -10
❗ 心态 < 20 → RE/MLE 崩溃，算法 -5
❗ 行动前检查消耗，避免属性归零
```

## 🎁 小技巧

```
✨ 夜晚学习效率更高，但消耗更大
✨ 人际关系高时，请教大佬成功率提升
✨ 购买「机械键盘」等装备有永久增益
✨ 写题解博客既能提升算法又能涨声望
✨ 适当「冥想」可以快速恢复心态
```
